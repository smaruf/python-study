## Key UI Features

### 1. **Intuitive Navigation**
   - Clear main menu with large, colorful buttons
   - Back buttons on all dialogs
   - Consistent layout across screens

### 2. **Parameter Organization**
   - Grouped by function (Wing, Fuselage, Tail, etc.)
   - Section headers with colored backgrounds
   - Logical flow from top to bottom

### 3. **Input Validation**
   - Default values provided for all fields
   - Real-time validation on generate
   - Clear error messages if invalid input

### 4. **Output Options**
   - Material selection dropdown
   - Checkboxes for output types
   - Directory picker for save location

### 5. **Professional Appearance**
   - Color-coded by aircraft type
   - Clean, modern design
   - Readable fonts and spacing
   - Emoji icons for visual appeal

### 6. **Responsive Design**
   - Scrollable forms for all screen sizes
   - Resizable windows
   - Maintains usability on smaller screens
