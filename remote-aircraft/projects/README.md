## Adding a New Project

1. Create a subdirectory under `projects/` named `<descriptive-slug>/`.
2. Add a `README.md` with overview, specs, BOM summary, firmware notes, and build steps.
3. Optionally add a `bom.md` (full Bill of Materials) and `firmware/` skeletons.
4. Update the table above with a one-line entry.
5. Reference the new project from the parent [`README.md`](../README.md) under
   the **Community Projects** section.
