# Level 0 - Beginner

Welcome to Level 0! This is where your automation journey begins.

## Learning Objectives

- Understand basic script structure
- Run your first automation scripts
- Learn basic input/output operations
- Get system information using scripts

2. **user_input.sh** - Learn to accept and use user input
3. **system_info.sh** - Display basic system information

## How to Run

### Bash (Linux/macOS)
```bash
cd bash
chmod +x *.sh
./hello_world.sh
./user_input.sh
./system_info.sh
```

### PowerShell (Windows/Cross-platform)
```powershell
cd powershell
.\hello_world.ps1
.\user_input.ps1
.\system_info.ps1
```

### Batch (Windows)
```cmd
cd batch
hello_world.bat
user_input.bat
system_info.bat
```

### Python
```bash
cd python
python hello_automation.py
python user_input.py
python system_info.py
```

## Key Concepts

### Output
- **Bash**: `echo "text"`
- **PowerShell**: `Write-Host "text"`
- **Batch**: `echo text`
- **Python**: `print("text")`

### Input
- **Bash**: `read variable_name`
- **PowerShell**: `$variable = Read-Host "prompt"`
- **Batch**: `set /p variable="prompt"`
- **Python**: `variable = input("prompt")`

### Variables
- **Bash**: `name="value"` and use with `$name`
- **PowerShell**: `$name = "value"`
- **Batch**: `set name=value` and use with `%name%`
- **Python**: `name = "value"`
