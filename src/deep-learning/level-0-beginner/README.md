# Level 0 - Beginner: Deep Learning Fundamentals

Welcome to Level 0! This level covers the essential foundations you need before diving into deep learning.

## Overview

This level focuses on:
- Python basics for machine learning
- NumPy arrays and tensor operations
- Data preprocessing and normalization
- Basic visualization techniques
- Mathematical foundations

## Prerequisites

- Basic Python knowledge
- High school level mathematics

## Modules

### 01_numpy_basics.py
Introduction to NumPy, the fundamental package for numerical computing in Python.

**Topics:**
- Creating and manipulating arrays
- Array indexing and slicing
- Broadcasting
- Mathematical operations
- Linear algebra basics

**Run:**
```bash
python 01_numpy_basics.py
```

### 02_tensor_operations.py
Understanding tensors and their operations - the building blocks of deep learning.

**Topics:**
- Tensor creation and initialization
- Tensor shapes and reshaping
- Element-wise operations
- Matrix multiplication
- Tensor reduction operations

**Run:**
```bash
python 02_tensor_operations.py
```

### 03_data_preprocessing.py
Learning how to prepare data for machine learning models.

**Topics:**
- Data normalization and standardization
- Handling missing values
- Train/validation/test splits
- Data augmentation basics
- Feature scaling

**Run:**
```bash
python 03_data_preprocessing.py
```

### 04_visualization.py
Visualizing data and results effectively.

**Topics:**
- Matplotlib basics
- Plotting distributions
- Visualizing multidimensional data
- Creating subplots
- Saving figures

**Run:**
```bash
python 04_visualization.py
```

## Learning Objectives

By the end of Level 0, you should be able to:
- Create and manipulate multi-dimensional arrays efficiently
- Understand tensor operations and broadcasting
- Preprocess data for machine learning
- Visualize data distributions and results
- Understand the mathematical foundations of ML

## Time Estimate

1-2 weeks, spending 1-2 hours per day

## Next Steps

Once you're comfortable with these fundamentals, proceed to [Level 1 - Basic](../level-1-basic/README.md) where you'll build your first neural networks!
